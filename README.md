# Web Chat

Demo: https://chat-lyart-gamma.vercel.app/

## Какие технологии используются:
- React.js
- Redux
- Redux-thunk
- React-hook-form/Yup
- Axios
- Socket.io-client
- Material UI/Emotion 

## Что реализовано:
- Авторизация, Регистрация (JWT)
- Добавление друзей (Invite)
- Список друзей
- Валидация форм (схема валидации через Yup)
- Индикаторы онлайн и не в сети
